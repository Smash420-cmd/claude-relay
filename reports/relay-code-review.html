high Store re-parses JSON from disk on every operation src/store.js:60–73 perf
Every getTasks(), addTask(), updateTask() calls load() which reads + parses relay-data.json from disk. Under scheduler tick pressure this is dozens of file reads per minute.
Cache the parsed db in a module-level variable; invalidate only on the fs.watch callback or after each write.
high Spawn env leaks all secret env vars except ANTHROPIC_API_KEY src/executor.js:85 security
Only ANTHROPIC_API_KEY is stripped before spawning Claude. AWS_SECRET_ACCESS_KEY, GH_TOKEN, DATABASE_URL and any other secrets in the shell env are passed to child processes.
Use a whitelist of safe env vars (PATH, APPDATA, HOME, TEMP, etc.) instead of a blacklist, or strip all known secret patterns.
